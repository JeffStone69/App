# XForge Trader v9.2 Beta

**Self-Improving Autonomous Trading Intelligence**  
Powered by Grok (xAI) • May 2026

## Quick Start (macOS)

1. Place `LOGO-SIM.jpeg` in the repository root.
2. Create a `.env` file from `.env.example` and add your `XAI_API_KEY`.
3. Double-click `launch.command`.
4. Choose option **2** (fast launch).
5. The app opens automatically in your browser at:  
   **http://127.0.0.1:7860**

## Features in v9.2 Beta
- Silent / hidden terminal launch
- Beautiful splash screen using your logo
- Large-text, dynamic fullscreen cyber UI
- Real-time watchlist + signals
- Strategy optimizer + paper trading
- Simulated trading history
- Self-Improvement Module (SIM) with Grok
- Full event logging to `XForge_Beta.log`

## Requirements
- Python 3.10+
- macOS (primary) or Windows
- Valid xAI API key
